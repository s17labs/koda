Editing made easy

Koda is a clean, lightweight text editor for Android.

Built for focused writing and coding sessions with multi-tab editing, smart file workflows, and a polished dark interface.

View on GitHub Download

Features

Multi-tab editing

Work with several files at once without breaking your flow.

File management

Create, open, save, and save-as with smart auto-naming.

Unsaved change detection

Get prompted before closing or switching away from modified files.

Customizable editor

Choose font family, size, line wrap, auto-save, and default save folder.

WakeLock support

Keep your screen on while writing or reviewing long documents.

Language support

Available in English and Slovak.

Tech

Android SDK 33

Built against modern Android APIs with a minimum SDK of 26.

Kotlin

Entire app written in Kotlin for concise and maintainable code.

Gradle Kotlin DSL

Project configuration uses Gradle with Kotlin-based build scripts.

Getting started

git clone https://github.com/s17labs/koda.git

Open the project in Android Studio, then build and run on a device or emulator.
